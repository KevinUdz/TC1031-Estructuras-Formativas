# Correciones realizadas a los entregables

Este repositorio contiene las correcciones realizadas a los tres entregables que inicialmente no lograron compilar. A continuación, explico de manera detallada qué fue lo que ocurrió y cuáles fueron los cambios que hice para que el código funcionara correctamente.


Al momento de hacer la entrega original, por un error de organización, utilicé un archivo main.cpp que no correspondía al ejercicio correcto. Esto provocó que los programas no compilaran, ya que el main no estaba alineado con las funciones, estructuras o lógica que sí eran parte del entregable. En pocas palabras, el problema no estaba en las funciones que desarrollé, sino en que usé un main equivocado, lo cual generó errores de compilación en los tres entregables.

Correcciones realizadas
1. Sustitución del archivo main.cpp

2. Ajustes menores en funciones

3. Verificación de compilación



El fallo original se debió a que tomé por error un main que no debía usar. Esto provocó que las funciones no coincidieran con lo que dicho main esperaba, lo cual generó errores de compilación.
Después de identificar el error, corregí el main para cada entregable y realicé pequeñas mejoras al código para asegurar su correcto funcionamiento. Agrdezco la segudna oportunidad profe y pido una disculpa por mi confusion.
